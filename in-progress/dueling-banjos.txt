Dueling Banjos – Instrumental

Section A (play through one time):

Guitar: slowly play 5 note G-chord, one note at a time
Banjo: response
Guitar: strum G-C-G
Banjo: response
Guitar: repeat strum G-C-G
Banjo: response
Guitar: treble string lick
Banjo: response
Guitar: repeat treble string lick
Banjo: response
Guitar: Yankee doodle went to town
Banjo: riding on a pony

Section B (play through twice):

Guitar: bass string G chord lick
Banjo: response
Guitar: bass string G chord lick
Banjo: response
Guitar: bass string C chord lick
Banjo: response
Guitar: bass string G chord lick
Banjo: response
Guitar: bass string D chord lick
Banjo: response
Guitar: strum G-C-G
Banjo: response
Guitar: repeat strum G-C-G
Banjo: response
Guitar: treble string lick
Banjo: response
Guitar: treble string lick
Banjo: response (this is omitted in the YT instructional, maybe on accident?)
Guitar (and banjo): Yankee doodle went to town, riding on a pony...then bass run up to C for rhythm guitar part
Guitar rhythm part:
 C C G G
 D D G G
 C C G G
 D D G G

Section C (ending, play once):
G G
NC (4 bars)
G G A/D G
